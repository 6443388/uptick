# Uptick Application

## Requirements
* CentOS v7
* NGINX for CentOS v7
* node.js v9.x.x
  * mssql ODB driver v3.3.0
  * express

## Setup & Configuration

### Multi Node
* Create 2x CentOS v7 Guest VMs
  * Install NGINX on Guest VM #1
  * Install node.js v9.x on Guest VM #2
  
### Single Node
Create a CentOS v7 Guest VM
  * Install NGINX 
  * Install node.js
